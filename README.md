# Titlu proiect: Get a Job
## Descriere: Joc de tip text-based RPG dezvoltat in C++, utilizand concepte de programare orientata pe obiecte (POO)

## Cerințe
- pentru fiecare cerință (sau subcerință) neîndeplinită se scade **1** punct
